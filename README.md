# Calculator

*A simple calculator in with Spring & VueJS*

## Get Started

```bash
git clone https://github.com/sunnybeta/calc
cd calc
docker compose up
```

Visit [localhost:4173/](https://localhost:4173)

## Stack

| Tech  | Framework |
| :---: | :---:|
| Front End | VueJS|
| Back End |  Spring|
| Containerization | Docker|
